To run the project you need to run the MainProgram.java
file in your IDE which you can find on this path:
MazeGen/program/src/main/java/com/example/program/control/MainProgram.java

If you need any help playing press the "Help" button in the game to see the rules and how to play.
In the settings page, you can also select if you want to play with your mouse or with the keyboard.

Our github link: https://github.com/AdrPou/MazeGen
